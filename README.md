# 🌲 TreeScribe
> A terminal-aesthetic, browser-based ASCII directory tree creator — no install, no backend, just open and use.

## Features
- Drag & drop folder import — drop any local folder to auto-generate its tree
- Visual tree builder — add, rename, drag-reorder, and delete nodes
- 4 tree styles: Classic, Rounded, Pure ASCII, Minimal
- Paste & import existing trees or path lists
- Copy to clipboard or download as .txt
- Terminal-inspired dark UI with CRT aesthetics

## Deploy to GitHub Pages
1. Put `index.html` in the root of your repo
2. Go to Settings → Pages → Deploy from branch → main
3. Done — live at `https://yourusername.github.io/treescribe`

## Local Use
Just open `index.html` in any modern browser. No build step, no dependencies.
